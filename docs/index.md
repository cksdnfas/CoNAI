# CoNAI 기능 맵

## 한눈에 보는 핵심

- 라이브러리 탐색과 검색: 프롬프트 기반 검색, 생성 기록 연결 확인
- 컬렉션 정리: 폴더 구조 기반 자동 그룹 정리, 프롬프트 그룹 재사용
- 이미지 분석과 자동 분류: 자동 태깅, 작가·스타일 추정
- AI 생성과 워크플로우: NAI·ComfyUI 생성, 재사용 가능한 워크플로우 실행
- 자동 수집과 자동화: 지정 폴더 자동 수집, 외부 도구·에이전트 연동
- 저장과 내보내기: 그룹 단위 내보내기, 메타데이터 수정 후 재저장
- 사용자 커스텀: 커스텀 테마, 저장 규칙 조정
