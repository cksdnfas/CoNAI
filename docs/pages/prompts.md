# Prompts

## 페이지 목적

프롬프트를 타입별로 모아 보고, 그룹화하고, 정리하는 페이지입니다.

## 이 페이지에서 할 수 있는 일

- positive, negative, auto 프롬프트 전환
- 프롬프트 검색
- 정렬 기준과 정렬 방향 변경
- 프롬프트 그룹 탐색
- 프롬프트 그룹 생성, 수정, 삭제, 순서 변경
- 프롬프트 그룹 import/export
- 프롬프트 요약 보기
- 프롬프트 수집 실행
- 단일 또는 다중 프롬프트 그룹 할당
- 프롬프트 삭제
- 프롬프트 복사
