# Upload

## 페이지 목적

파일 업로드와 메타 추출 실험을 함께 처리하는 페이지입니다.

## 이 페이지에서 할 수 있는 일

- 이미지와 비디오 업로드
- 업로드 진행률 확인
- 업로드 전 저장 옵션 확인
- 단일 이미지 메타 미리보기
- 태거 자동 추출 실행
- Kaloscope 추출 실행
- 통합 추출 실행
- WebP 변환 다운로드
- 메타 수정 파일 다운로드
- 드래그 앤 드롭 업로드/추출
