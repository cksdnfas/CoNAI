# Groups

## 페이지 목적

이미지를 커스텀 그룹과 감시폴더 그룹으로 나눠 탐색하고 관리하는 페이지입니다.

## 이 페이지에서 할 수 있는 일

- 커스텀 그룹과 폴더 그룹 전환
- 루트 그룹과 하위 그룹 탐색
- 그룹 생성, 수정, 삭제
- 전체 자동수집 실행
- 감시폴더 재구축 실행
- 현재 그룹 다운로드
- 선택 이미지 다운로드
- 선택 이미지를 커스텀 그룹에 추가
- 커스텀 그룹에서 선택 이미지 제거
- 그룹 내부 이미지 탐색과 선택
