# Image Generation

## 페이지 목적

이미지 생성과 생성 이력, 그리고 워크플로우 기반 생성 구성을 한곳에서 다루는 페이지입니다.

## 이 페이지에서 할 수 있는 일

- NAI 생성
- ComfyUI 생성
- Wildcard 생성
- 생성 이력 확인
- 생성 컨트롤 새로고침
- 모바일에서 컨트롤 드로어 열기
- Workflow 탭에서 그래프 기반 작업 진입
- `/graph` 경로 접근 시 workflow 탭으로 이동

## Workflow 탭에서 할 수 있는 일

- 워크플로우 목록 탐색
- 워크플로우 편집 진입
- 노드와 연결 관리
- 워크플로우 실행, 재실행, 재시도, 취소
- 실행 결과와 산출물 확인
- 폴더 단위 브라우즈와 관리
