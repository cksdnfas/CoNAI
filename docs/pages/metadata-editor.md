# Metadata Edit

## 페이지 목적

기존 이미지의 메타데이터를 검토하고, 다운로드용 재작성 또는 실제 운영 저장을 수행하는 페이지입니다.

## 이 페이지에서 할 수 있는 일

- 메타 rewrite draft 확인
- 포맷 선택
- 메타 수정 파일 다운로드
- 실제 운영 파일과 라이브러리 DB에 저장
- 이미지 상세 페이지로 돌아가기

## 연결된 핵심 시스템

- [미디어 메타데이터 엔진](/systems/media-metadata)
- [생성 저장 파이프라인](/systems/generation-pipeline)

## 주의 포인트

- 정적 이미지 파일만 저장 편집을 지원합니다.
- 실제 저장은 운영 파일 교체와 보존 전략과 연결됩니다.
