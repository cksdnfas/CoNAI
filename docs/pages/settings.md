# Settings

## 페이지 목적

폴더, 화면 테마, 자동 추출, 메타 추출, 이미지 저장 정책을 관리하는 제어실입니다.

## 이 페이지에서 할 수 있는 일

- watched folder 설정 관리
- appearance 설정 저장, 초기화, 취소
- appearance preset slot 저장
- 테마 패키지 import/export
- 커스텀 폰트 업로드
- tagger 설정 저장
- Kaloscope 설정 저장
- tagger 의존성 확인
- 자동 테스트 대상 찾기 또는 랜덤 선택
- tagger 자동 테스트 실행
- Kaloscope 자동 테스트 실행
- metadata 추출 설정 저장
- image save 설정 저장
