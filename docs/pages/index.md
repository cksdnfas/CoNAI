# 페이지 맵 개요

이 섹션은 실제 라우트 기준으로 CoNAI 화면을 정리합니다.
다만 문서의 우선순위는 여전히 **핵심 시스템 먼저, 화면 설명은 그 다음**입니다.

## 주요 페이지

- [Groups](/pages/groups)
- [Prompts](/pages/prompts)

## 페이지 읽는 법

각 페이지 문서는 아래 순서로 정리합니다.

1. 페이지 목적
2. 사용자가 할 수 있는 일
3. 연결된 핵심 시스템
4. 앞으로 더 문서화할 세부 항목
