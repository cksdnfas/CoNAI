# Home

## 페이지 목적

CoNAI의 기본 이미지 피드입니다.
전체 라이브러리를 가장 빠르게 훑고, 검색 결과를 바로 확인하는 첫 화면입니다.

## 이 페이지에서 할 수 있는 일

- 전체 이미지 피드 보기
- 검색 칩 기반 결과 확인
- 무한 스크롤로 다음 이미지 로드
- 이미지 다중 선택
- 선택 이미지 다운로드
- 선택 이미지를 그룹에 추가
- 이미지 상세 페이지로 이동
