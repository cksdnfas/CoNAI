# Release Notes

## Version 26.4.13 (2026-04-13)

v26.4.13은 이미지 상세/모달 사용성 개선, 메타데이터 저장 안정화, 권한/계정 관리 정비, 생성 큐/워크플로우 실행 상태 정합성 개선, 월페이퍼 편집 UX 정리를 중심으로 한 업데이트입니다.

---

### 이미지 상세 / 모달 UX 개선

이미지 보기 흐름을 더 빠르고 가볍게 다듬었습니다.

- 이미지 모달에 `L / M / S` 3단계 뷰 모드 추가
- 중간 모드는 프롬프트 중심의 가벼운 상세 보기로 정리
- 초경량 모드는 풀스크린 뷰어 중심으로 단순화하고 이전 / 다음 탐색 유지
- 웹 표시용 이미지를 전역적으로 thumbnail-first 규칙으로 통일
- 이미지 상세 / 모달에서 휠 줌, 드래그 이동, 회전, 모바일 핀치 줌을 지원하는 비파괴 뷰어 추가
- 상세 / 모달 비디오 재생 UI를 공용 플레이어로 정리하고 자동 재생 + 루프 재생 흐름 개선
- 무한 로드 리스트와 이미지 모달 탐색을 더 자연스럽게 이어지도록 동기화
- 무거운 보조 쿼리와 스트립 렌더를 지연해 full 모달 오픈 체감 성능 개선

---

### 메타데이터 저장 / 유사도 개선

편집 결과의 신뢰성과 유사 이미지 확인 흐름을 함께 손봤습니다.

- 이미지 메타 저장 시 DB 반영과 실제 파일 교체가 함께 이뤄지도록 저장 흐름 안정화
- 기존 운영 파일은 보관 경로로 이동하고, 새 파일이 active 파일로 승격되도록 정리
- 메타 편집 revision 기록 구조를 추가해 이후 복원/이력 기반 확장 준비
- `Auto prompt` 복사 액션을 `GENERAL` 헤더 쪽으로 이동해 맥락 정리
- 유사 이미지 / 텍스트 유사 패널을 더 가볍고 일관된 스타일로 재구성
- 이미지/텍스트 탭 점수 배지를 공용 컴포넌트로 통합
- 중복 이미지 판정을 `pHash + dHash + aHash` 기반으로 보강해 false positive 완화
- 유사도 설정 패널 용어, 툴팁, 가중치/임계값 표기를 더 직관적으로 정리

---

### 계정 / 권한 관리 개선

로컬 로그인과 보안 설정 흐름을 좀 더 실사용 중심으로 정리했습니다.

- 로컬 로그인 사용 중 로그인된 상태라면 상단 헤더에서 계정 팝업을 열고 현재 계정 확인 및 로그아웃 가능
- 보안 설정 화면을 built-in 그룹 위주 카드 묶음에서 권한 그룹 중심 흐름으로 재구성
- 시스템 그룹(`anonymous`, `guest`, `admin`)은 보호하면서 커스텀 권한 그룹 생성 / 수정 / 삭제 지원
- 권한 그룹 안에서 페이지 접근 권한을 함께 편집하도록 구조 정리
- 그룹별 멤버 추가 / 제거 흐름 지원
- 계정 목록에서 현재 소속 그룹을 함께 확인 가능

---

### 생성 큐 / 워크플로우 실행 정합성 개선

큐 표시와 실제 실행 상태가 어긋나던 부분을 보수적으로 정리했습니다.

- 백엔드 재시작 시 중단된 graph workflow 실행이 `queued` / `running` 상태로 영구 잔류하지 않도록 startup recovery 추가
- 재시작으로 끊긴 queued / running 실행은 명시적인 재실행 필요 메시지와 함께 `failed`로 정리
- 실행 이력을 참조하는 schedule도 ghost active 상태에 묶이지 않도록 후속 상태 정리 보강
- generation queue 위치 / ETA 계산이 실제 dispatcher routing 해석과 같은 기준을 사용하도록 정렬
- background queue 읽기 / 운영 액션 권한을 generation 화면 기준에 맞춰 더 엄격하게 정리
- queue 패널 / 헤더 위젯의 상태 라벨, ETA, 타임스탬프, mutation 후 refresh 흐름을 공용 helper로 정리

---

### 월페이퍼 편집 / 런타임 정리

월페이퍼 편집기와 실제 보기 사이의 어색함을 줄였습니다.

- 위젯 공통 표시 옵션으로 `경계선 표시` 토글 추가
- 기본값은 체크 해제 상태로 두고 실제 런타임 보더 노출을 더 깔끔하게 제어
- 월페이퍼 위젯 편집 패널 구조를 다시 정리
- 커스텀 easing 편집기, 프리셋, 미리보기, 애니메이션 제어를 확장하고 안정화
- 런타임 이미지 표시와 전환 관련 세부 동작을 더 다듬음
- 기존 `queue` 중심 용어가 실제 graph workflow 실행 상태를 가리키던 부분은 사용자 표시 문구를 `워크플로 상태` / `실행 펄스` 중심으로 정리
