# Release Notes

## Version 26.4.5 (2026-04-05)

v26.4.5는 메타데이터 저장 안정화, ComfyUI 모듈 저장 흐름 보강, Workflows 탭 UX 정리, 와일드카드/문서 정리를 중심으로 한 업데이트입니다.

---

### 이미지 메타데이터 저장 안정화

이미지 메타데이터 편집 기능의 저장 흐름을 실제 사용 기준에 맞게 보강했습니다.

- 메타데이터 저장 시 DB 반영뿐 아니라 실제 운영 파일 교체 기반 흐름을 정리
- 기존 파일은 RecycleBin 보존 대상으로 처리하여 복원 가능성 확보
- 메타데이터 편집 revision 기록 구조 추가
- 관련 저장 동작 및 문구를 현재 동작 기준에 맞게 정리

---

### ComfyUI 모듈 저장 흐름 개선

ComfyUI 워크플로우를 재사용 가능한 모듈로 저장하는 흐름을 더 자연스럽게 만들었습니다.

- ComfyUI 워크플로우에서 모듈 저장 진입점 추가
- 워크플로우 목록과 상세 화면에서 모두 모듈 저장 가능하도록 정리
- 모듈 라이브러리 모달 내부의 불필요한 접기 UI 제거
- 모듈 저장 버튼 위치를 기존 NAI 흐름과 더 비슷하게 정리

---

### Workflows 탭 / Module Graph UX 정리

`generation?tab=workflows` 및 module graph 편집 흐름의 사용성을 개선했습니다.

- 실행 상세 모달에서 섹션 이동 버튼을 헤더 하단 고정 위치로 이동
- 워크플로우 편집 지원 패널도 같은 방식으로 섹션 점프 버튼 정리
- 스크롤 시 중요한 이동 버튼이 가려지지 않도록 개선
- 워크플로우 폴더 생성 / 할당 / 수정 / 삭제 흐름 보강
- 저장된 워크플로우 목록, 실행 결과, 최종 결과 표시 구조 정리

---

### Wildcards / 기타 UI 개선

와일드카드 및 일부 보조 UI를 더 보기 쉽게 다듬었습니다.

- `generation?tab=wildcards` 사이드바 상단 액션 버튼 정렬 정리
- 삭제 버튼에 위험 액션 색 포인트 추가
- 일부 LoRA auto collect / inline media preview 연동부 정리

---

### 문서 정리

프로젝트 내 오래된 문서도 함께 정리했습니다.

- 오래된 릴리즈 노트 정리
- 과거 작업 계획 문서 대량 정리
- 현재 기준 유지 가치가 낮은 구형 문서 제거

---
