# Release Notes

## Version 26.3.26

26.3.26은 `c8855bd0c13b470f38ed97cb82eced96b0eb2acb` 이후 누적된 대형 UI 개편, 메타데이터 편집/저장 흐름 확장, 탐색기 구조 재정비, 런타임/빌드 환경 정리를 한 번에 묶는 알파테스트 기준 버전입니다.

> 상태: **AlphaTest 최신 작업 기준 정리본**

---

### 핵심 변경 사항

#### 1) 이미지 탐색/상세/선택 UX 대형 개편

갤러리와 상세 보기 흐름이 전반적으로 재구성되었습니다.

- 이미지 리스트를 가상화 기반 구조로 재정비해 대량 미디어 탐색 성능을 개선
- 스마트 선택 모드, 배치 다운로드, 선택 액션 바 추가
- 이미지 상세 페이지를 섹션 단위로 분리하고 유사 이미지 설정/탭 흐름 강화
- 공용 이미지 상세 모달, 썸네일 스트립, 독립 스크롤, 모바일 액션 바 고정/실측 보강 등 상세 뷰어 UX 개선
- 비디오/GIF가 리스트/선택 흐름에서 더 자연스럽게 동작하도록 보정

#### 2) 그룹/탐색기 구조 재편

자동 그룹과 커스텀 그룹을 기존 단순 목록에서 탐색기형 구조로 옮기는 작업이 진행되었습니다.

- 그룹 탐색 전용 페이지와 공용 탐색 레이아웃 도입
- 자동 폴더 그룹 / 커스텀 이미지 그룹을 explorer 스타일로 전환
- 모바일 그룹 화면에서 bottom-sheet 기반 이미지 패널/생성 컨트롤 도입
- 최근 작업트리 기준으로 감시 폴더 실제 구조를 더 직접 반영하는 자동 폴더 그룹 정비가 포함됨

#### 3) 메타데이터 편집/재기록 워크플로우 확장

업로드 단계와 라이브러리 편집 단계 모두에서 메타데이터 편집 기능이 크게 확장되었습니다.

- 업로드 시 WebP 변환 및 메타데이터 재기록 흐름 추가
- WebP 메타데이터 추출/검증 및 공용 EXIF/XMP writer 도입
- 업로드 화면에서 메타데이터 재작성 컨트롤과 프리뷰 자동 추출 추가
- 라이브러리 이미지 메타 편집 v1 추가
- 메타 저장 시 DB 갱신뿐 아니라 새 active 파일 생성, revision 기록, 교체 저장 흐름까지 확장

#### 4) Appearance / Theme 시스템 확장

외형 커스터마이징이 단순 테마 선택을 넘어 실사용 편집기 수준으로 확장되었습니다.

- appearance theme 설정, 고급 워크플로우, 프리셋 슬롯 도입
- 내장 폰트 자산 패키징 및 저장 시 불필요 자산 정리
- 상태 표면/카드 톤 일관성 강화
- related image 카드 appearance 옵션, 모바일 카드 열 수, 폰트 weight 프리셋 추가
- 최근 작업 기준으로 appearance 편집 섹션 분할, UI scale / text scale 분리 등 세부 설정도 보강

#### 5) 검색 / 프롬프트 / 설정 화면 정리

주요 화면을 기능별로 다시 나누고 공용 구조를 늘려 유지보수성을 개선했습니다.

- 전역 갤러리 검색 UX와 검색 히스토리 흐름 재구축
- 프롬프트 탐색기 페이지와 prompt similarity/detail 탭 추가
- 설정 페이지를 탭/카드/도메인 단위로 분리
- 프론트 API 모듈과 백엔드 라우트를 책임별로 분리해 코드 구조를 재정비
- 공용 폼 컨트롤, explorer 패턴, shared section 컴포넌트 재사용 범위 확대

#### 6) 런타임/호환성/운영 정비

알파테스트 실사용을 위한 환경 정비도 크게 진행되었습니다.

- Node 24 호환을 위한 `better-sqlite3` 및 네이티브 의존성 정리
- frontend build toolchain을 Vite 8 기준으로 업데이트
- 기본 Upload 폴더/감시 폴더 안정화, 폴더 설정 flicker 감소
- backup source ingest watcher 추가 및 최근 작업트리 기준 backup target 경로 안내/생성 UX 보완
- 빠른 시작용 배치 런처 정리

### 현재 작업트리까지 포함되는 26.3.26 마무리 항목

이 버전에는 커밋 이력 외에 현재 알파테스트 작업트리에 포함된 다음 조정도 함께 반영됩니다.

- 자동 폴더 그룹 서비스가 감시 폴더의 실제 디렉터리 구조를 더 직접 반영하도록 보정
- backup source watcher / 생성 폼 / 카드 UX 후속 수정
- 그룹 페이지/트리/이미지 섹션/드로어 흐름 후속 정리
- 홈 검색 UI 및 이미지 리스트 상호작용 후속 수정
- 실행 배치 파일 이름 정리 (`RUN_CoNAI_Start.bat` 기준)
- 불필요/구버전 문서 일부 정리
