# 홈/이미지 보기

홈은 CoNAI에 등록된 이미지와 비디오를 보는 기본 피드입니다. 감시 폴더 스캔, 업로드, 생성 결과 저장이 끝나면 대부분 여기에서 확인합니다.

## 관련 페이지

| 목적 | 경로 |
| --- | --- |
| 홈 피드 | `/` |
| 이미지 상세 | `/images/:compositeHash` |
| 메타데이터 편집 | `/images/:compositeHash/metadata` |
| 그룹 관리 | `/groups` |
| 업로드 | `/upload` |

## 표시되는 항목

홈은 등록된 미디어를 카드 목록으로 보여줍니다.

- 정적 이미지
- GIF 같은 애니메이션 이미지
- MP4/WebM/MOV/AVI/MKV 계열 비디오
- 처리 중인 항목의 상태 배지
- 공개/권한 설정에 따라 흐림 처리되거나 숨겨진 항목

상단 상태 줄에는 표시 수, 로드 수, 전체 수, 숨김 수가 나타납니다. 목록은 처음에 일부만 불러오고, 하단의 **더 보기** 또는 스크롤 로딩으로 다음 묶음을 불러옵니다.

## 기본 사용 흐름

1. `/`로 이동합니다.
2. 카드가 보이지 않으면 감시 폴더 스캔 또는 업로드가 먼저 끝났는지 확인합니다.
3. 이미지를 클릭해 상세를 엽니다.
4. 필요한 항목을 선택합니다.
5. 선택 바에서 다운로드, 그룹 추가, 삭제를 실행합니다.

로그인한 일반 화면에서는 클릭 시 상세 모달을 열고, 직접 링크나 공개 화면에서는 상세 페이지로 이동할 수 있습니다.

## 카드 열 수 조정

홈 우측 하단의 열 수 컨트롤에서 카드 밀도를 조정합니다.

- 열 수 증가: 한 화면에 더 많은 항목 표시
- 열 수 감소: 개별 이미지를 크게 보기
- 초기화: 기본 열 수로 복귀

이 설정은 홈 목록 전용입니다. 그룹 페이지의 열 수 설정과 별도로 동작합니다.

## 선택 작업

이미지를 선택하면 하단 선택 바가 열립니다.

| 작업 | 설명 |
| --- | --- |
| 다운로드 | 선택한 원본/렌더 가능 파일을 내려받습니다. |
| 그룹 추가 | 선택 이미지를 커스텀 그룹에 넣습니다. |
| 선택 삭제 | 삭제 권한이 있을 때만 표시됩니다. 삭제 보호 설정에 따라 RecycleBin으로 이동할 수 있습니다. |
| 선택 해제 | 현재 선택을 비웁니다. |

선택 삭제는 되돌리기 비용이 큰 작업입니다. 운영 환경에서는 [설정 전체 지도](./SETTINGS_OVERVIEW.md)의 삭제 보호와 [데이터 경로와 백업](./DATA_PATHS_AND_BACKUP.md)을 먼저 확인하세요.

## 상세로 넘어가기

상세 화면에서는 다음을 확인합니다.

- 큰 미리보기
- 복합 해시
- 크기, 파일 크기, 원본 경로
- 모델, 스텝, 샘플러, 시드 등 생성 파라미터
- 긍정/부정/자동/작가 프롬프트
- 중복 이미지
- 이미지 유사도
- 프롬프트 유사도

자세한 내용은 [이미지 상세와 유사도 검사](./IMAGE_DETAIL_AND_SIMILARITY.md)를 참고하세요.

## 홈에 이미지가 없을 때

1. `/settings` → 감시 폴더에서 폴더가 등록되어 있는지 확인합니다.
2. 전체 스캔을 실행합니다.
3. 스캔 로그에서 실패 항목을 확인합니다.
4. 제외 확장자/패턴이 너무 넓지 않은지 확인합니다.
5. 현재 계정에 홈 보기 권한이 있는지 `/access` 또는 설정 → 보안에서 확인합니다.
6. 업로드 파일이라면 `/upload` 결과에서 실패 목록을 확인합니다.
