# MCP와 자동화 인터페이스

## 역할

CoNAI를 화면 밖에서 호출하게 해주는 외부 자동화 접점입니다.
AI 클라이언트, 에이전트, 자동화 스크립트가 CoNAI 기능을 사용하려면 이 계층이 필요합니다.

## 이 시스템이 하는 일

- 이미지 검색과 프롬프트 검색 요청을 외부에서 받습니다.
- 생성 호출을 자동화 경로로 연결할 수 있습니다.
- 서버 기능 일부를 구조화된 도구 형태로 제공합니다.
- 향후 워크플로우 자동화와도 연결될 수 있습니다.

## 연결되는 대표 화면

- 직접 보이는 전용 UI 페이지는 적지만, 프로젝트 가치 설명에서 매우 중요합니다.
- Settings와 백엔드 서버 운영 문맥에서 자주 언급됩니다.

## 사용자 입장에서 보이는 효과

- CoNAI를 단순 로컬 앱이 아니라 자동화 가능한 플랫폼으로 확장합니다.
- 외부 AI 도구가 같은 데이터와 기능을 재사용할 수 있습니다.

## 문서화 포인트

- 이 기능은 제품 소개 첫머리에 나와야 합니다.
- 일반 사용자에게는 안 보여도, CoNAI 차별점 설명에는 핵심입니다.
