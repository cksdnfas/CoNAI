# 감시폴더 등록과 스캔

## 역할

실제 파일이 CoNAI 라이브러리로 들어오는 첫 관문입니다.
watched folder를 기준으로 파일을 빠르게 등록하고, 기존 파일 상태를 다시 확인합니다.

## 이 시스템이 하는 일

- 감시폴더 아래 파일을 탐색합니다.
- 신규 파일을 빠르게 1차 등록합니다.
- 기존 파일의 변경 시각, 크기, 상태를 갱신합니다.
- 폴더 구조를 자동 그룹 구조와 연결합니다.
- 스캔 진행률과 오류를 추적합니다.

## 중요한 특징

- 빠른 등록 단계에서는 최소 정보부터 확보합니다.
- 삭제 상태 파일과 활성 파일을 구분해 다룹니다.
- 이후 메타 추출이나 상세 분석 단계로 이어질 수 있습니다.

## 연결되는 대표 화면

- Settings > Folders
- Groups의 폴더 그룹 뷰
- Home 피드

## 사용자 입장에서 보이는 효과

- 새 파일이 라이브러리에 들어옵니다.
- 감시폴더 재구축이나 재스캔이 가능한 이유입니다.
- 폴더 기반 분류가 화면에 살아나는 기반입니다.
